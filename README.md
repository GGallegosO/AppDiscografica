🎸 Sistema de Gestión de Eventos - Santo Tomás
Proyecto final para la asignatura de Programación, desarrollado bajo arquitectura de 3 Capas.

🚀 Características
CRUD Completo: Crear, Leer, Actualizar y Eliminar eventos.

Arquitectura: Separación de responsabilidades (UI, Negocio y Datos).

Base de Datos: SQL Server gestionado a través de Entity Framework.

Validaciones: Control de reglas de negocio en la capa correspondiente.

🛠️ Tecnologías
C# / .NET

Windows Forms

SQL Server
